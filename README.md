## Setup & Installation

Make sure you have the latest version of Python installed.

```bash
git clone <repo-url>
```
Requirements (Import)

- flask
`pip install flask `

- Flask-SQLAlchemy
`pip install Flask-SQLAlchemy `

- flask-login
`pip install Flask-Login `

## Running The App

```bash
python main.py
```

## Viewing The App

Go to `http://127.0.0.1:5000`
